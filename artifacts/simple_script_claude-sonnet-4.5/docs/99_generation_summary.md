## 🎯 プロジェクト概要

### 目的

CSVファイル（date, value形式）を読み込んで統計情報を計算し、グラフを含むHTMLレポートを生成するコマンドラインツールを提供する。

### 主要機能

1. ✅ **CSVデータ読み込み**: pandas による高速な読み込みとバリデーション
2. ✅ **統計計算**: 平均、中央値、最小値、最大値、標準偏差の自動計算
3. ✅ **データ可視化**: matplotlib による時系列グラフとヒストグラム生成
4. ✅ **HTMLレポート生成**: 統計情報とグラフを含む見やすいレポート
5. ✅ **コンソール出力**: 実行時の簡易サマリー表示

### 技術スタック

- **言語**: Python 3.11
- **データ処理**: pandas 3.0.0
- **可視化**: matplotlib 3.10.8
- **CLI**: argparse（標準ライブラリ）
- **テスト**: pytest 9.0.2 + pytest-cov 7.0.0
- **品質ツール**: black 26.1.0, mypy 1.19.1, ruff 0.15.0

### アーキテクチャ

**単一ファイル構成** (`analyze.py`)
- シンプルで理解しやすい
- 6つの主要関数で構成
- 型アノテーション完備
- 包括的なエラーハンドリング

## 🚀 使用方法

### インストール

```bash
# 仮想環境の作成とパッケージインストール
uv venv
uv sync
source .venv/bin/activate
```

### 実行

```bash
# サンプルデータで実行
python analyze.py sample_data.csv

# 独自のCSVで実行
python analyze.py your_data.csv -o my_report.html
```

### テスト

```bash
# テスト実行
pytest

# カバレッジ付きテスト
pytest --cov=. --cov-report=html
```

### 品質チェック

```bash
# コードフォーマット
black analyze.py

# 型チェック
mypy analyze.py --ignore-missing-imports

# Linting
ruff check analyze.py
```
