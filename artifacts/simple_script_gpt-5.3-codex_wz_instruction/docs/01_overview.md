# 01. 要件概要整理

## 目的
CSV ファイルを読み込み、基本統計とグラフを作成し、HTML レポート（`report.html`）として出力する Python 製のコマンドラインスクリプトを提供する。

## 想定ユーザー
- 手元の時系列データ（`date`, `value`）を簡易分析したい利用者
- Python を直接書かずに、CLI でレポート作成したい利用者

## スコープ（このシステムが提供すること）
### 機能要件
1. **CSV 読み込み**
   - 入力 CSV を読み込み、必要カラムの存在と型を検証する
2. **基本統計の計算**
   - `value` の平均・中央値・最大・最小を算出する
3. **データ可視化**
   - `date` を横軸、`value` を縦軸とした時系列グラフを生成する
4. **HTML レポート生成**
   - 統計情報とグラフを含む `report.html` を出力する
5. **コンソールサマリー表示**
   - 実行結果（統計サマリー、件数、出力先など）を標準出力に表示する

### 入力仕様
- CSV カラム
  - `date`: 日付（`YYYY-MM-DD` 形式）
  - `value`: 数値

### 出力仕様
- `report.html`: 統計情報 + グラフを含む HTML レポート
- コンソール出力: 簡易サマリー

## 非機能要件（明文化）
- CLI として実行できる（`argparse`）
- エラー時は原因が分かるメッセージを出力して非 0 終了
- 小規模〜中規模 CSV（目安: 数万行程度）で実用的に動作

## 依存管理方針（本プロジェクトの前提）
要件原文には `requirements.txt` が記載されているが、以降フェーズでは **pyproject.toml による依存管理**を採用する（`[project] dependencies` と `optional-dependencies.dev` を使用）。
