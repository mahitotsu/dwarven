# プロジェクト概要 (Overview)

## プロジェクト名
Data Analysis Tool

## 目的
CSV形式の時系列データ（日付と数値）を読み込み、統計分析を行い、可視化グラフを含むHTMLレポートを自動生成するコマンドラインツール（CLI）を提供する。

## 主要機能
1.  **データ読み込み**: CSVファイルを読み込み、適切な型（日付、数値）に変換する。欠損値のチェックを行う。
2.  **データ分析**: 以下の統計情報を計算する。
    *   平均値 (Mean)
    *   中央値 (Median)
    *   最大値 (Max)
    *   最小値 (Min)
3.  **データ可視化**: 時系列データの折れ線グラフを生成する。
4.  **レポート生成**: 分析結果とグラフを埋め込んだHTMLファイルを生成する。
5.  **CLIインターフェース**: 入力ファイルパスや出力先を指定できるコマンドライン引数を提供する。

## 利用シーン
ユーザーが日次の数値データ（売上、気温、アクセス数など）を持っおり、素早く傾向を把握してレポートとして共有したい場合に使用する。
