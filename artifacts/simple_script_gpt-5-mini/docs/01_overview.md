# 概要

目的: CSV（date, value）を読み取り、基本統計とグラフを生成し、HTML形式のレポート（report.html）とコンソールサマリーを出力するコマンドラインツールを提供する。

スコープ:
- 入力: 日付（YYYY-MM-DD）と数値（value）を含むCSV
- 出力: report.html（統計＋グラフ）、コンソールサマリー
- 対象ユーザー: データ解析の自動化を望むエンジニアやアナリスト

成功基準:
- 正常なCSVを解析して正しい統計値を出力できること
- グラフがレポートに埋め込まれて表示されること
- CLIで入力ファイル指定と出力先を指定できること

制約・前提:
- 日付はYYYY-MM-DD形式を想定（不正な形式はエラーまたはスキップ）
- 大規模データ（数百万行）向けの最適化は初期フェーズでは対象外

依存関係（候補）:
- pandas
- matplotlib または plotly
- jinja2（HTMLテンプレート）
- argparse（標準ライブラリ）

運用:
- ローカルCLI実行を想定。CIではユニットテストと静的解析を実行する。
