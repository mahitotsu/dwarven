# データモデル設計

入力スキーマ（CSV）:
- date: 文字列（YYYY-MM-DD）→ pandasでdatetimeに変換
- value: 数値（float）

DataFrame上の列（内部表現）:
- date: datetime64[ns]
- value: float64
- year: int（解析用に追加可能）
- month: int（解析用に追加可能）

前処理:
- 欠損値: valueが欠損の場合、デフォルトは行を除外。オプションで補完（線形補間など）を検討。
- 異常値: 基本は検出して警告。必要に応じて閾値によるフィルタを実装。
- 日付パース失敗: ログ出力して該当行をスキップまたはエラー終了（オプションフラグで挙動切替）。

出力アーティファクト:
- report.html: 統計概要（平均/中央値/最大/最小）、グラフ（時系列等）を含むHTML
- console summary: 標準出力へ簡易統計を表示
- 中間ファイル（任意）: 生成したグラフ画像（PNG/SVG）や集計CSV

メタデータ:
- 実行時に使用したパラメータ（入力ファイル名、処理日時、使用した閾値など）をレポートに記載
