# プロジェクト概要

## プロジェクト名
CSV Data Analyzer

## 目的
CSVファイルから数値データを読み込み、統計分析を行い、視覚的なレポートを生成するコマンドラインツールを提供する。

## 主要機能

### 1. CSVファイルの読み込み
- YYYY-MM-DD形式の日付と数値データを含むCSVファイルを入力として受け取る
- 不正なデータ形式のエラーハンドリング

### 2. 統計情報の計算
以下の統計指標を計算：
- 平均値（mean）
- 中央値（median）
- 最大値（max）
- 最小値（min）
- 標準偏差（std）
- データ件数（count）

### 3. データ可視化
以下のグラフを生成：
- 時系列折れ線グラフ（日付 vs 数値）
- ヒストグラム（数値の分布）
- 統計サマリーの表形式表示

### 4. レポート生成
- HTML形式のレポートファイルを出力
- 統計情報、グラフ、データサマリーを含む
- コンソールに簡易サマリーを表示

## ユースケース

### 基本的な使用方法
```bash
python -m csv_analyzer analyze input.csv
```

### オプション指定
```bash
python -m csv_analyzer analyze input.csv --output custom_report.html
```

## 成功基準
1. 正常なCSVファイルを正しく読み込み、処理できる
2. 統計情報が正確に計算される
3. グラフが正しく生成される
4. HTMLレポートが読みやすい形式で出力される
5. エラーが適切にハンドリングされる

## 制約事項
- Python 3.11以上が必要
- 入力CSVは UTF-8 エンコーディング
- メモリに収まるサイズのデータを想定（大規模データは対象外）
