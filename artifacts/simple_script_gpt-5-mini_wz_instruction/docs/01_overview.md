# 概要

目的: CSVデータを読み込み、集計・可視化し、HTMLレポートとコンソールサマリーを出力するコマンドラインツールを作成する。

スコープ:
- 入力: date（YYYY-MM-DD）、value（数値）を持つCSV
- 出力: report.html（統計+グラフ）、コンソールサマリー
- 最低限のCLI（ファイル指定、出力先、可視化ライブラリ選択オプション）

非機能要件:
- 再現性のあるコマンドライン操作
- ログ出力（INFO/ERROR）
- テスト可能なモジュール設計
- 小〜中規模データ（数万行）を想定した実行性能

成功基準:
- 与えたCSVから正しい統計値が得られ、HTMLレポートにグラフと要約が含まれること
- CLIで入出力を指定できること
- ユニットテストが主要ロジックをカバーしていること
